A React app that helps you write haikus by counting syllables in real-time and providing instant feedback.

## Features

- Real-time syllable counting (5-7-5 pattern)
- Save haikus to local storage
- View and delete saved haikus
- Accessible keyboard navigation
- Responsive design

## Tech Stack

- React
- Vite
- Vitest (testing)
- Local Storage API

## Run Locally

```bash
npm install
npm run dev
```

## Tests

```bash
npm test
```
